# 1-Minute Demo Script

## Opening

This project is not trying to build another personal AI assistant.
It is trying to build an organizational intelligence layer for Feishu.

## Step 1: show the world model

Open the sample data and explain that the system models:
- goals
- projects
- tasks
- risks
- DRIs
- signals

## Step 2: run risk scan

Show how the system identifies blockers, overdue items, missing DRIs, and unhealthy projects.

## Step 3: run DRI suggestion

Show how the system recommends a directly responsible individual instead of leaving coordination to meetings and manual escalation.

## Step 4: generate weekly brief

Show the management summary output.

## Step 5: show Feishu output path

Show the generated Docs-style output and Feishu Docs write-intent artifact.

## Closing line

So this project is not just a CLI. It is a prototype of how Feishu could host an organizational intelligence layer that helps teams move work forward.
